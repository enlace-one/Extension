Welcome to ! We aim to offer useful features ran entirely locally with optional encryption. You'll notice this extension has three views, each with its own features. This is the options view or extension tab view (). If you press the icon on the upper right, you will see the popup view (). Finally, if you open the sidebar you will see the sidebar view ()

Set Up

Keyboard shortcuts are essential. We have taken the privelege of making some default shortcuts available to you, but we recommend you visit extension shortcut page to add one for the popup view and edit the others if you'd like.
    1. Go to "[browser]://extensions/shortcuts".
        Example:"edge://extensions/shortcuts"
    2. Set the "Activate the extension" shortcut
        Example:"Alt+E"

It is nice to be able to run shortcuts globally (while using an app other than your browser). For the shortcut to work globally, it has to follow this pattern: Ctrl+Shft+[0-9].
    Example: Ctrl+Shift+1

Some combinations have had issues working, so if you encounter that, just try a different one.

More Info

Data storage is set to sync using the browser's built in sync storage method. So it may work across your devices. If the information itself is necessary, KEEP A COPY SOMEWHERE ELSE. This is designed for convienence, not long-term storage.

Data is encrypted with AES256 encryption using a key generated via the salted password you enter. Without the password, it is irrecoverable. There is currently no way to recover your password. If that data is set to be encrypted, then the data will always be encrypted before stored. (See settings)

Some third party libraries are used. easyMDE which is a fork of simpleMDE which are both based on CodeMirror are used for page notes. Additionally, crypto-js is used for encryption. If I am missing any references, please email me. easyMDE and crypto-js are downloaded and included with this, but CodeMirror is retrieved from a CDN (I did not want to modify the easyMDE code to change that).

Permissions and Privacy

This extension does not collect any personal data. It does not track your browsing history. In fact, everything runs locally on your machine.

This extension requires the following permissions (which are granted by default on install):
"<all_urls>" is required for the feature to work.
"storage" is required for the and features to work.
"offscreen" is required for the feature to work.
"clipboardWrite" and "clipboardRead" are required for the feature to work.
"sidePanel" is required for the feature to work.
"tabs" is required for the feature to work.
"userScripts" is required for the feature to work.
"contextMenus" has been required in the past and there are plans in the future too.
