How does the URL pattern work?
The extension checks if the current link contains the url pattern. It accounts for the wildcard '*' which matches anything and "|" which means "OR".
For example, "https://*.example.com|https://example.com" will match:
  • https://example/com/path/one
  • https://something.example.com
  • https://somethingelse.example.com/path/two
